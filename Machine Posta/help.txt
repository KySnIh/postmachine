Машина Поста

Команды:
	> N		- сдвиг вправо, переход на N строку
	< N		- сдвиг влево, переход на N строку
	1 N		- поставить метку, переход на N строку
	0 N		- убрать метку, переход на N строку
	? N, M	- если метки нет, то переход на N строку, иначе переход на M строку
	!		- завершение программы
	S		- запуск скрипта
	V +-N	- перемешение указателя на ленте
	P [0,1]	- первоначальные значения ленты
